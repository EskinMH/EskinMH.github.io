CONCLUSIONES

En esta actividad logramos aprender a como programar un botón para ocultar un menú, también logre recordar el como usar bootstrap y lo útil que es para nuestro futuro.

En Javascript existe un concepto llamado evento, que no es más que una notificación de que acaba de ocurrir una operación interesante, generalmente relacionada con la navegación del usuario por la página. Los eventos permiten al usuario enviar información al programa. JavaScript define una serie de eventos que permiten una interacción completa entre el usuario y la página web/aplicación. Presionar una tecla constituye un evento, al igual que hacer clic o mover el mouse, seleccionar un elemento de formulario, cambiar el tamaño de la ventana del navegador, etc. JavaScript le permite asignar un rol a cada evento. Entonces, cada vez que ocurre un evento, JavaScript realiza la acción correspondiente. Estas funciones se denominan "event handler" en inglés y, a menudo, se traducen como "manejadores de eventos"
